
# 📄 Document Signature Application

A full-stack PDF Document Signature Application built using React, Node.js, Express, MongoDB, and PDF-lib.

## 🚀 Features

- User Registration & Login (JWT Authentication)
- Upload PDF Documents
- View PDF Documents
- Draw Digital Signature
- Type Signature
- Place Signature Anywhere on PDF
- Download Signed PDF
- MongoDB Database Integration
- Responsive UI
- Android App Support using Capacitor

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer
- PDF-lib
